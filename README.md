## Overview

Servidio is a simple JavaScript Frontend Library of Services.

You will find Axios CRUD Services & String Services.

## Content

Axios part :  
-   **getData(url)**  
-   **postData(url, data)**  
-   **patchData(url, data)**  
-   **putData(url, data)**  
-   **deleteData(url)**

String part :  
-   **checkString(str, type)**  
-   **rewriteString(str, type)**
